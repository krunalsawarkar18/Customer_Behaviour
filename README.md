# Customer Shopping Behavior Analysis

## Overview
This project is a complete data analyst portfolio case study focused on customer shopping behavior, revenue optimization, and segmentation for a retail dataset. It combines exploratory data analysis, data cleaning, SQL analytics, and cross-database integration to deliver clear business insights and actionable recommendations.

## Project Objective
- Analyze customer purchase behavior and identify trends in revenue, discounts, subscription status, and product preferences.
- Build a repeatable data workflow using Python, pandas, and SQL.
- Demonstrate strong data analyst skills in data cleaning, EDA, database integration, and storytelling with visual reports.

## Files Included
- `customer_shopping_behavior.csv` — Raw dataset used for analysis.
- `Customer_Shopping_Behavior_Analysis.ipynb` — Python notebook with data cleaning, feature engineering, and database loading scripts.
- `customer_behavior_sql_queries.sql` — SQL queries answering business questions related to revenue, discounts, customer segmentation, and top products.
- `customer_behavior_dashboard.pbix` — Power BI dashboard file for visualization and reporting.
- `Business Problem  Document.pdf` — Business problem statement and project objectives.
- `Customer Shopping Behavior Analysis.pdf` — Project summary and findings presentation.
- `Customer-Shopping-Behavior-Analysis.pptx` — PowerPoint presentation for stakeholder communication.
